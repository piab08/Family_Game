# Family_Game
This game consists of 2-4 players and each player rolls a die and performs a task

# Rules 
Each person rolls a die and performs a task once the task is completed the die is rolled by another person and the position keeps updating of each player. Atlast, the player who reaches to 30 first wins the game

# How to run the program
Clone the project in a specific location 
Then install the modules required from requirements.txt
`streamlit run family_game.py`
